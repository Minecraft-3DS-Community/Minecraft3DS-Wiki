Unistore Manual

Installing the Unistore

Your console needs to have custom firmware (CFW) to continue. If you do not have it already, you can get it by following this guide: https://3ds.hacks.guide/.

If you do not have the Universal Updater, then please download and install it from here: Universal Updater

Make sure any other Homebrew is up to date, mainly Luma3DS: Luma3DS

Finally, in order for mods to work, Game Patching needs to be enabled in the Luma Config.

  • Power off the console
  • Turn on the console while holding Select to open the Luma Config
  • Use the D-Pad to navigate to "Enable game patching" and press A (it should have an [X] next to it)
  • Press the Start button to exit and save.

Then you can install the Unistore by opening the Universal Updater, and going to Settings > Select Unistore > Add, and selecting the Minecraft 3DS Community Unistore in the list of recommended Unistores.

Using the Unistore

Installing a mod is simple, just press A on the mod you want, select the sub-mod, and choose the region of your game. Then you click "Confirm" and go to the queue and accept any pop-ups after reading them. The mod you just installed should show up in the game, no extra steps needed. To delete mods, find the "Customization Removal" mod, and choose which type of mod you want to remove. "All content" removes any modifications to the game, it does not delete your worlds or any save data. If you install another unistore with the mc3ds unistore it should be in the "choose unistore" menu.

Mods aren't working?

These are common fixes for Mods not working for Minecraft 3DS.

DLC/Update + Game:

Sometimes if you have a region-free game or DLC/update and a USA/EUR/JPN copy of Minecraft 3DS. They will cause mods to not fully work or at all. Make sure you have everything matching the same region, and nothing is amiss/mismatched.

LCL:

  • Power off the console
  • Turn on the console while holding Select to open the Luma Config
  • Use the D-Pad to navigate to "Enable game patching" and press A (it should have an [X] next to it)
  • Press the Start button to exit and save.

If the option doesn't show an [X] on Game Patching, then it's disabled and needs to be enabled.

Download Mods for your game's region:

Make sure you installed the correct region of the mod.

Combatting Mods:

Some Mods such as Skyblock+ (Plus) and Amplified World Generation combat each other, and cannot be installed together on the same system.

Game Version:

Almost all of the mods for Minecraft New Nintendo 3DS Edition were made for Patch v1.9.19.

Incorrectly Made Mods:

Some mods are made incorrectly and some functions might not work if the developer did not do rigorous testing.

Mods not installing:

If your mods aren't installing from the Minecraft 3DS Unistore, you should try reinstalling the unistore.
