How to Edit Json Files

Learn how to edit json files using these json commands for mc3ds, each command will contain a description and snippet of code

minecraft:is_baby

This command is used for component groups which reflect baby varients.

Code snippet

"minecraft:is_baby": { },

minecraft:scale

This command is used to rescale a compontent groups model, such as babies.

Code snippet

"minecraft:scale": {"value": X},
